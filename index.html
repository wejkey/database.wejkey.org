Frequently Asked Questions

Common questions about the database

What is Wejkey Database?

+

Wejkey Database is a repository for all my projects. (Created primarly for Minecraft plugins) Detailed documentation, version history, changelogs, screenshots, and direct download links for each project/plugin (if any).

Does this database have a higher priority?

+

Yes, when an update is released for a project, this database is updated first and only then are the other sites.

Where can I report bugs or request features?

+

For now, this is only possible at contact@wejkey.org. (TIP: If you dont want to use your real email, use temporary email address)

What does "Beta" mean?

+

The database itself is in beta, meaning the website is still being refined and new features are being added.
